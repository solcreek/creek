## Quick Start

```bash
npm create astro@latest my-app -- --template basics
cd my-app
npm install
npx creek deploy
```

## Configuration

No configuration needed. Creek auto-detects Astro.

| Setting | Default |
|---------|---------|
| Build command | `npm run build` |
| Output directory | `dist` |
| Render mode | Static (default Astro output) |
